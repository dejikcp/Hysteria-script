# Hysteria-script

Hysteria 一键脚本，支持IPv4、IPv6 VPS

## 使用方法

```shell
wget -N https://raw.githubusercontents.com/Misaka-blog/Hysteria-script/master/hysteria.sh && bash hysteria.sh
```

快捷方式 `bash hysteria.sh`
